# AI Voice Assistant using Python

## Overview

This project is a simple AI Voice Assistant developed using Python.  
The assistant can recognize voice commands, respond using speech, open websites, provide date and time information, search Wikipedia, and maintain command history.

## Features

- Speech Recognition
- Text-to-Speech Conversion
- Open YouTube
- Open Google
- Tell Current Time
- Tell Current Date
- Wikipedia Search
- Play Music Folder
- Save User Command History
- Clear History Command

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- sounddevice
- scipy
- wikipedia
- webbrowser
- datetime
